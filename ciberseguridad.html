Recreación de página

¿Cuál fue el objetivo del desafío?

Este proyecto consistió en la recreación visual de una página profesional existente, con el objetivo de practicar y demostrar habilidades en maquetación, componentes reutilizables y estilado moderno utilizando React y Tailwind CSS.

La consigna principal fue lograr una réplica lo más fiel posible en términos de estructura, colores, tipografía y distribución de elementos, manteniendo la semántica HTML y aplicando buenas prácticas de desarrollo frontend.

Principales enfoques del desarrollo:

División modular en componentes: Cada sección de la página —como navegación, encabezado, tarjetas de información o botones— fue implementada como componente React independiente, lo que favorece la reutilización y el mantenimiento del código.

Uso de Tailwind CSS: Se empleó esta librería de utilidad para aplicar estilos precisos de forma rápida, facilitando el control visual exacto de cada bloque (espaciado, colores, tipografía y responsive).

Fidelidad visual: Se trabajó cuidadosamente en la disposición y apariencia de los elementos para lograr una réplica exacta del diseño original, incluso en sus detalles más mínimos como iconografía, jerarquías visuales y proporciones.

Este ejercicio permitió afianzar la lógica de diseño basada en componentes, aplicar estilos de forma eficiente y entrenar el ojo para la precisión visual, habilidades clave en el desarrollo web moderno.

Tecnologías usadas:

  • Next.js

    Framework basado en React que permite renderizado del lado del servidor (SSR), generación de sitios estáticos (SSG) y manejo avanzado de rutas.

  • React

    Biblioteca JavaScript para construir interfaces de usuario interactivas y reutilizables.

  • TypeScript

    Superset de JavaScript que añade tipado estático, ayudando a detectar errores en tiempo de desarrollo.

  • Tailwind CSS

    Framework de utilidades CSS que permite diseñar interfaces modernas de forma rápida y eficiente directamente en el HTML.

  • Chart.js

    Biblioteca JavaScript para la creación de gráficos dinámicos y visualmente atractivos.

  • React Chart.js 2

    Wrapper que permite integrar los gráficos de Chart.js dentro de componentes de React de manera sencilla.

  • Zustand

    Librería ligera y minimalista para el manejo del estado global en aplicaciones React.

  • React Icons

    Conjunto de librerías de iconos populares integradas en componentes de React para una fácil implementación.
