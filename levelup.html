Noticias, análisis de videojuegos y Hardware

¿Cuál fue el objetivo del desafío?

Este proyecto consistió en el desarrollo completo de una página web informativa e interactiva llamada Level Up, centrada en noticias del mundo gaming y hardware. El objetivo fue practicar maquetación moderna, estructura semántica y diseño adaptable a distintos dispositivos.

Se buscó lograr una interfaz clara, bien distribuida y visualmente atractiva, donde cada sección ofreciera información relevante de forma ordenada, aplicando buenas prácticas en desarrollo web.

Principales enfoques del desarrollo:

Maquetación con HTML semántico: Se utilizó una estructura clara y lógica mediante etiquetas apropiadas, mejorando la accesibilidad y la organización del contenido.

Estilos con CSS, SASS y uso puntual de Bootstrap: La implementación de Bootstrap fue mínima, limitándose a algunos elementos base, mientras que los estilos principales fueron desarrollados con CSS puro y SASS, aprovechando variables, anidamiento y modularización.

Diseño completamente responsive: La adaptabilidad a distintos tamaños de pantalla fue trabajada manualmente, garantizando una experiencia óptima tanto en dispositivos móviles como de escritorio.

Este proyecto permitió reforzar conocimientos clave en maquetación, estructura visual y adaptabilidad, integrando tecnologías fundamentales para el desarrollo frontend profesional.

Tecnologías usadas:

  • HTML

    Lenguaje de marcado utilizado para estructurar el contenido de la página de forma semántica y accesible.

  • CSS

    Hoja de estilos en cascada empleada para definir el diseño visual de los elementos y personalizar la interfaz.

  • SASS

    Preprocesador de CSS que permite escribir estilos más organizados y mantenibles, utilizando funciones como variables y anidamiento.

  • Mixins

    Funcionalidad de SASS que permite reutilizar bloques de estilos con parámetros personalizados, facilitando un diseño más dinámico.

  • Extends

    Característica de SASS usada para compartir propiedades comunes entre selectores, reduciendo la repetición de código.

  • Bootstrap

    Framework de componentes y utilidades que fue implementado mínimamente para apoyar el diseño, priorizando el desarrollo manual y personalizado.
