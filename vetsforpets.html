Acercando la solución para la salud de tu mascota

VetsForPets es una plataforma web desarrollada para facilitar la conexión entre propietarios de mascotas y centros veterinarios cercanos, brindando una solución integral para la gestión de su salud y bienestar.

Funcionalidades principales:

Registro de Mascotas: Permite a los usuarios registrar a sus mascotas con datos relevantes como raza, edad, historial médico y necesidades específicas, generando un perfil completo para su seguimiento.

Geolocalización de Veterinarias: A través de un mapa interactivo, los usuarios pueden ubicar clínicas veterinarias cercanas en tiempo real, optimizando tiempos de búsqueda y acceso a servicios.

Agendamiento de Turnos: El sistema permite solicitar y gestionar citas médicas en línea de forma rápida, evitando llamadas telefónicas y asegurando una mejor organización para el usuario y el profesional.

Botón de Emergencia: Funcionalidad clave para situaciones críticas que conecta al usuario con un profesional veterinario mediante chat en vivo, brindando asistencia inmediata.

Planes de Membresía: Los usuarios pueden acceder a planes exclusivos que ofrecen beneficios como descuentos en consultas, vacunas y medicamentos, incentivando la atención preventiva y regular de las mascotas.

Tecnologías Utilizadas

Backend:

Node.js: Entorno de ejecución de JavaScript para el desarrollo del backend.

NestJS: Framework estructurado para construir aplicaciones escalables y de alto rendimiento.

TypeScript: Lenguaje con tipado estático que mejora la calidad del código.

PostgreSQL: Base de datos relacional utilizada para almacenar la información de usuarios, mascotas y veterinarias.

TypeORM: Herramienta ORM que facilita la interacción con la base de datos PostgreSQL.

JWT: Implementación de autenticación y autorización mediante tokens seguros.

Swagger: Generación automática de documentación para la API.

WebSocket.io: Comunicación en tiempo real empleada en el sistema de chat de emergencias.

Stripe: Integración para procesar pagos de forma segura, incluyendo planes de membresía.

Nodemailer: Envío automático de correos electrónicos para notificaciones y confirmaciones.

Cloudinary: Almacenamiento y gestión de imágenes y archivos en la nube.

OAuth con Google: Autenticación mediante terceros para mejorar la experiencia del usuario.

Frontend:

Next.js: Framework basado en React para crear aplicaciones web optimizadas y escalables.

React: Librería de JavaScript para el desarrollo de interfaces de usuario interactivas.

TypeScript: Mejora la robustez y escalabilidad del código en el frontend.

Leaflet: Librería para integrar mapas interactivos y funcionalidades de geolocalización.

CSS: Lenguaje de estilos utilizado para definir el diseño visual de la aplicación.

Tailwind CSS: Framework de utilidades CSS que permite un desarrollo rápido y responsivo.

WebSockets: Comunicación en tiempo real aplicada al sistema de emergencias.

Calendly: Herramienta para agendar citas de manera sencilla e integrada.

OAuth con Google: Autenticación de usuarios de manera rápida y segura mediante cuentas de Google.
