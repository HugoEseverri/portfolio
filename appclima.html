¿De qué trata Weather App?

Weather App es una aplicación web desarrollada con Angular, que permite a los usuarios consultar el estado del clima en tiempo real en cualquier ciudad del mundo. Su interfaz moderna, construida con SCSS, ofrece una experiencia visual agradable y totalmente responsiva.

La aplicación se apoya en una arquitectura escalable basada en Standalone Components de Angular, lo que mejora su mantenibilidad y rendimiento general.

Principales características:

Búsqueda por ciudad: Los usuarios pueden consultar el clima ingresando el nombre de cualquier ciudad, accediendo instantáneamente a información actualizada.

Visualización detallada: Se muestran datos relevantes como temperatura actual, humedad y condiciones atmosféricas, todo presentado de forma clara y atractiva.

Diseño moderno y adaptable: Gracias a SCSS, Weather App se adapta a todo tipo de dispositivos, brindando una experiencia coherente desde móviles hasta pantallas de escritorio.

Arquitectura modular: Utiliza Angular con Standalone Components para mantener una estructura organizada y preparada para futuras ampliaciones.

Weather App combina funcionalidad, diseño y escalabilidad para ofrecer una herramienta confiable y práctica que permite a los usuarios mantenerse informados sobre el clima de forma ágil y precisa.

Tecnologías usadas:

  • Angular

    Framework moderno para construir aplicaciones web dinámicas, basado en componentes modulares y TypeScript.

  • TypeScript

    Lenguaje de programación tipado que mejora la calidad y mantenibilidad del código JavaScript.

  • SCSS

    Preprocesador CSS que permite escribir estilos más estructurados, reutilizables y fáciles de mantener.

  • OpenWeatherMap API

    API externa utilizada para obtener información del clima en tiempo real a partir del nombre de la ciudad.
